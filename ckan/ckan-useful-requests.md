action/package_list
    params: 
    Return a list of the names of the site's datasets.
    e.g. `curl -d '{"limit":"1"}' <API URL>/action/package_list`

action/package_show
    params:
    Return the metadata of a dataset and its resources.



REST requests (nicer) (prefix rest):

`package/<package_name>`
    Return the metadata of a dataset and its resources.
